Reescrevendo um dos principais artigos que fizeram o Solister.top ser criado, mas que deixei inativo por um tempo. Trago aqui dois patches para instalar o SimCity 2000 nos Windows mais novos.

Onde ocorre o problema

Se você possui um Windows (x86) (32-Bits), você provavelmente consiguirá instalar o SimCity 2000 sem nenhum problema, porém após instalar e quando for salvar ou abrir uma cidade, o jogo fecha sem deixar nenhum tipo de erro.

Se você possui um Windows (x64) (64-Bits), o problema já começa logo na instalação, pois o Windows bloqueia o instalador e nem se quer é possível instalá-lo (Nem mesmo se copiar os arquivos e você instalar por si mesmo).

Então para isso, aqui está um mini tutorial de como instalar o jogo.

Instalando o jogo

Se você usa um sistema x86 (32-Bits), você pode simplesmente pular esta etapa pois você consiguirá instalar o jogo sem problemas, caso não consiga, talvez deva dar uma pequena lida aqui.

No meu caso, estou usando o SimCity 2000 Special Edition em Português. Nele há três versões do mesmo jogo. Uma de MS-DOS, outra de Windows 3.1 e a que irei utilizar, de Windows 95.

Para instalar o jogo, você deverá usar um instalador alternativo. Você pode encontrá-lo aqui. (Não foi feito por mim). Você só precisa seguir o tutorial, ou se preferir, pode ir testando as opções até achar alguma que funcione para você. Esse patch também arruma o jogo para Abrir/Salvar o jogo, porém o executável do jogo fica todo em inglês (Os textos do jogo ficam dentro do .exe), para isso siga o próximo tutorial para deixá-lo em português.

Patch para Abrir/Salvar Cidades

Se você usa um Windows (x86) (32-Bits), seu tutorial começa aqui, e se você usa um Windows (x64) (64-Bits), você ainda precisa usar este segundo patch.

Como explicado acima, após o jogo estar instalado o jogo fecha sem dar nenhum erro ao salvar/abrir alguma cidade. Felizmente, é possível arrumar isso através de um patch. Você só precisa substituir o .exe do jogo por esse aquivo (Não foi feito por mim, mas foi traduzido por mim). Após isso o jogo deverá funcionar normalmente em qualquer Windows acima do Windows 7.
