Este é um projeto que eu havia feito que dá ao Windows 10 um tema mais clássico similar aos Windows 95 e 98. Eis algumas screenshots:

Windows 98
Windows 95

Instalação:
  1. Baixe os temas, e extraia cada um em uma pasta separada. Faça um ponto de restauração se sentir inseguro.
  2. Instale o programa Classic Shell que está dentro da pasta. Se ele começar a te irritar quando liga o Windows, aperte Ctrl + Shift + Esc para abrir o task manager e vá na aba inicializar e desative-o.
  3. Usando a barra de pesquisa dentro do programa procure por Customize Taskbar. Haverá uma sub-seção, procure por Taskbar Opacity e coloque 0, e Taskbar color e coloque 382000. Isso fará o iniciar quase invísivel.
  4. Depois vá para Start Menu Settings e ponha Classic Style e clique para trocar o logo do Windows. O logo antigo se encontra na pasta Images.
  5. Vá no meio do Iniciar, pressione botão direito e clique configurações. Lá selecione a opção de icones pequenos. Aproveitando-se das configurações, aproveite para alterar o Wallpaper e a Lock Screen, com as imagens que se encontram também na pasta Images. Note que o Wallpaper possui uma linha bege, isso servirá para poder fazer o menu iniciar corretamente, pois o Windows 10 não suporta essa cor, nem mesmo com hacking. Se o Wallpaper não condizer com sua resolução procure algo similar no Google, mas terá de fazer a faixa bege por conta própria.
  6. Abra a pasta Ribbon Disable, e abra o aplicativo conforme os bits de seu sistema (32 ou 64).
  7. Para finalizar você pode trocar os ícones e os sons do Windows 10 pelos do Windows 95/98.
